Lav din egen widget til din kunde

Michael 2014/09/02

Find din functions.php i temaet, du arbejder med. I det her tilfælde ubercoool-2014-widget. Og ca. på linje 14 indsætter du: 


function ubercoool_widgets_init() {
	register_sidebar( array(
		'name'          => __( 'Top Widget', 'ubercoool' ),
		'id'            => 'top-widget',
		'description'   => __( 'Top Widget vil kunne ses lige over dine posts', 'ubercoool' ),
		'before_widget' => '<aside id="top-widget" class="top-widget-wrapper">',
		'after_widget'  => '</aside>',
		'before_title'  => '<h1 class="top-widget-title">',
		'after_title'   => '</h1>',
	) );
}
add_action( 'widgets_init', 'ubercoool_widgets_init' );

I den nye fil i det her tema, sidebar.php, indsætter du den her stump, der forholder sig til, om brugeren har oprettet noget i din top-widget - og hvis ja, så afvikles det:


if ( ! is_active_sidebar( 'top-widget' ) ) {
	return;
}
?>

	<div class="top-widget-wrapper widget-area clear" role="complementary">
		<?php dynamic_sidebar( 'top-widget' ); ?>
	</div>

Prøv at gå til dit dashboard i wordpress. Klik på Appearance > Widgets. Her kan du se, at du har en lang række mulige widgets, og til højre har du så den widget, vi arbejder på nu. Prøv at klik-og-trække 'Calendar' over i din 'Top Widget' og giv slip. Klik derefter på 'Save' eller 'Gem'.

Men prøv at se front-end delen og reload dit site. Der er ingen top-widget endnu. Vi skal lige fortælle Wordpress, hvor den skal sættes ind. Vi kan placere den, hvor vi har lyst, men i det her tilfælde vil placere den i headeren lige over vores posts:


<?php get_sidebar('top-widget'); ?>

Fino. Det virker. Men det ser ikke kønt ud. Åben din style.css i temaet, og indsæt koden her på ca. linje 192:


#top-widget {
	background: none repeat scroll 0 0 #FFFFFF;
    border: 1px solid rgba(190,199,199,0.7);
    border-radius: 2px 2px 2px 2px;
    margin: 1em 0 3em;
    padding: 15px;
    position: relative;
    -moz-box-shadow:    inset 1px 1px 1px rgba(20,40,60,0.3);
    -webkit-box-shadow: inset 1px 1px 1px rgba(20,40,60,0.3);
    box-shadow:         inset 1px 1px 1px rgba(20,40,60,0.3);
}
#top-widget:after {
    border-color: white transparent;
    border-style: solid;
    border-width: 10px 10px 0;
    bottom: -10px;
    content: "";
    display: block;
    right: 40px;
    position: absolute;
    width: 0;
    -moz-box-shadow:    inset 0 0 2px #000000;
    -webkit-box-shadow: inset 0 0 2px #000000;
    box-shadow:         inset 0 0 2px #000000;
}
#top-widget:before {
    border-color: rgba(190,199,199,0.7) transparent;
    border-style: solid;
    border-width: 11px 11px 0;
    bottom: -11px;
    content: "";
    display: block;
    right: 39px;
    position: absolute;
    width: 0;
}
#top-widget h1 {
	font-style: oblique;
	font-size: 22px;
}

Det var dét. Gå tilbage til dit Wordpress dashboard og klik på 'Calendar' i din 'Top Widget'. Klik på delete. Træk nu i stedet 'Recent posts' over i din 'Top Widget' og klik på 'Save' eller 'Gem'. På at reloade din front-end del af dit Wordpress site, og nu kan du, hvor nemt det bil blive for din kunde at indsætte widgets efter eget valg, der hvor du har gjort det muligt i din kode.
